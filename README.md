### 2. Short Description / Purpose

The ElectroDrive Insights Dashboard offers an interactive analytical view of electric vehicle trends. It allows users to explore how factors such as EV type (BEV/PHEV), state-level adoption, model year, vehicle make, and CAFV eligibility impact the distribution and growth of electric vehicles.
This dashboard supports researchers, policymakers, and industry analysts in understanding EV market behavior and making data-driven decisions
